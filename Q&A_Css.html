What is a clear property?

The clear property in CSS is used to control the behavior of elements when they encounter floated elements. Floated elements are typically aligned to the left or right using the float property, and the clear property is used to prevent the element from being positioned next to a floated element, ensuring that it is positioned below the float.
The clear property is primarily used to control how elements interact with floated elements, ensuring that they are not placed next to or overlapping floats. It is essential for managing layout, especially when working with floats.

What is :root in css?

The primary use of :root is to define custom properties of CSS variables that can be accessed and reused throughout the CSS. Because :root is the highest-level element, variables defined here are available globally.
It is often used for defining global CSS variables and applying styles with higher specificity than the html selector.

What is d.f b/w the transitions vs transforms vs animations?

In CSS, transitions, transforms, and animations are techniques used to create interactive and visually appealing effects, but they each serve different purposes.
Transitions create smooth changes between two states of an element.
The transform property allows us to visually manipulate an element by rotating, scaling, translating, skewing it without changing its position of the html document.
And An Animations lets an element graduallychange from one style to another.

What is a matrix in transforms?

Matrix () is a function in CSS transforms that allows you to apply complex 2D transformations (scaling, rotating, skewing, translating) with a single operation.

Diff b/w css and css3?

CSS: Earlier versions of CSS (like CSS1 and CSS2) were monolithic, meaning they were developed as a single specification covering all aspects of styling.
CSS3: Introduced a modular approach, where the entire specification is broken into smaller modules. Each module deals with a specific aspect of styling (like backgrounds, borders, animations, flexbox, etc.). This made development faster and allowed browsers to adopt features incrementally.
CSS3 Features:
*Selectors
*Box Model
*Backgrounds and Borders
*Transitions and Animations
*Flexbox and Grid Layout
*Media Queries
